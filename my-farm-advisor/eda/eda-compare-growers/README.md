# eda-compare-growers

Cross-grower EDA subskill comparing field boundaries, CDL/cropland, and weather across
the three Assignment 2 growers (Illinois, Iowa, Nebraska).

- **Script**: `data-pipeline/src/scripts/eda/eda_compare_growers.py`
- **Outputs**: 11 PNG plots + 2 CSV tables under `$DATA_PIPELINE_DATA_ROOT/data-pipeline/growers/derived/reports/eda_compare_growers/`
- **Report**: `report.html` (open in browser, images in `report_images/`)
- **Soil**: Not analyzed (`--skip-soil` used throughout)
- **Guide**: `GUIDE.md` for usage and workflow details
